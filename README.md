# WeOrganize.ai

AI-powered agency website for WeOrganize - helping businesses transform with intelligent automation solutions.

## Live Site

**Production:** https://weorganize-main.vercel.app

## Tech Stack

- **React** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **shadcn/ui** - UI components

## Getting Started

```sh
# Clone the repository
git clone https://github.com/mopshyai/weorganize.ai.git

# Navigate to project directory
cd weorganize.ai

# Install dependencies
npm install

# Start development server
npm run dev
```

## Deployment

This project is deployed on Vercel with automatic deployments on push to `main`.

To deploy manually:
```sh
vercel --prod
```

## Project Structure

```
src/
├── components/    # Reusable UI components
├── pages/         # Page components
├── hooks/         # Custom React hooks
├── lib/           # Utility functions
└── App.tsx        # Main application entry
```
